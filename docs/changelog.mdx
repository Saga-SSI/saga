<Update label="May 26, 2026">
  ## New features

  **Work kanban board.** Saga now ships with a collaborative work board as the primary app experience. Capture ideas, vote on what to build next, and assign teammates to move items forward. Learn more in the [Dashboard](/platform/dashboard) overview.

  **Sidebar navigation.** The app now has dedicated sections for **Home**, **Tribes**, **Village**, and **Work**. Signed-in users land on Home by default so it's easier to orient yourself when you open Saga.

  ## Updates

  **Refined Saga wordmark.** The Saga logo now uses a regular weight everywhere — across the navbar, sidebar, and auth screens — for a cleaner, more consistent brand feel.
</Update>
