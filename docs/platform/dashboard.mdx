# Dashboard

The dashboard is Saga's primary workspace at `/dashboard`.

## Features

- **Cursor Agent** — floating window (bottom-right) that runs the Cursor SDK against your local repository. Ask about code, architecture, or next steps.
- **Docs Assistant** — floating window (bottom-left) powered by Mintlify. Ask product and documentation questions with cited sources.

## Usage tips

1. Open `/dashboard` after starting the dev server.
2. Use **Cursor Agent** for codebase questions and implementation help.
3. Use **Docs Assistant** for Saga product docs and onboarding questions.

Both agents stream responses in real time.
